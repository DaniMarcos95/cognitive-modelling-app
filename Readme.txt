Follow this instruction to build EasyChords on a device:

1. First you need to install AudioKit in the project folder. For that run in a terminal on the project folder the following commands:
	
	1.1 (sudo) gem install

	1.2 (sudo) gem install -n /usr/local/bin cocoapods

	1.3 (sudo) pod init (may be not necessary)

	1.4 (sudo) pod install

2. Ater installing AudioKit, open 'Concentration.xcworkspace' in XCode.

3. Build the application in the selected device.
